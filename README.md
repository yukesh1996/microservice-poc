# node-microservice-poc

Example project for:

- Micro-service APIs
- Docker & Docker Compose
- EXPRESS API GATEWAY
- Node
- TypeScript
- Express
- PassportJS Auth
- MongoDB


## Getting Started
### Pre-reqs
- [Docker](https://docs.docker.com/engine/installation/)
- [Docker Compose](https://docs.docker.com/compose/)

### Running
Install and start docker by `docker-compose up`.
